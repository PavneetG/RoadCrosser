---------------------------------------------------------------------------------------------
------------------------- 1. RUNNING THE PROGRAM --------------------------
---------------------------------------------------------------------------------------------

In order to run the program you can either open the Launch.java file
and click run, or you can use the RoadCrosser.exe provided. The
login page should appear. If you have an account proceed to login,
or you can create a new account.

---------------------------------------------------------------------------------------------
----------------------------- 2. FILE DESCRIPTIONS ------------------------------
---------------------------------------------------------------------------------------------

LoginMenu.java
----------------------
The code for the login menu. Manages user accounts & loads user data.

HelpMenu.java
---------------------
The code for the help menu. Displays useful information about the game.

MainGame.java
----------------------
The code for the main game. Manages the gameplay, scoring, random 
generating, and everything surrounding making the game work.

AchievementsMenu.java
-----------------------------------
The code for the achievements menu. Manages achievement progress, 
and allows user to claim rewards.

HighscoresMenu.java
-------------------------------
Code for the highscores page. Sorts and displays highscores from all users.

MainMenu.java
----------------------
Code for the main menu. Connects all aspects of the program together, 
providing buttons to open other menus.

CharactersMenu.java
------------------------------
Code for the characters menu. Allows user to purchase and select characters 
to use in-game.

EndScreen.java 
----------------------
Code for the game over screen. Displays relevant user score information 
when the game ends.

PauseMenu.java
------------------------
Code for the pause menu. Allows user to pause the game, as well as restart or quit.
